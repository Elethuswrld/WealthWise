# **App Name**: WealthWise

## Core Features:

- Net Worth Overview: Aggregates and displays the user's net worth, including cash, crypto, stocks, forex, and other assets. Data fetched from Firestore.
- Income vs Expenses: Compares the user's income and expenses over time.  Data stored in Firestore.
- Portfolio Allocation: Calculates and displays the user's portfolio allocation as percentages in the UI, sourcing the data from Firestore.
- Transaction History: Logs user transactions, storing data to Firestore.
- Monthly Performance: Show monthly profit/loss with data derived from transaction history stored in Firestore.
- AI Insights: Generates personalized insights and recommendations based on the user's financial data such as pointing out overspending using a tool

## Style Guidelines:

- Primary color: Deep teal (#2A9D8F) for trustworthiness and stability.
- Background color: Light gray (#F0F4F3), offering a clean and unobtrusive backdrop.
- Accent color: Muted Blue (#264653) for highlighting key interactive elements.
- Body font: 'Inter', a sans-serif font that is clean and very readable.
- Headline font: 'Space Grotesk', a sans-serif font which pairs well with Inter and gives a clean technology forward appearance.
- Use minimalist, outline-style icons to represent different financial categories and actions.
- Dashboard layout inspired by Notion and TradingView.
- Subtle transitions and animations for data updates.
